Since some users are already familiar with the old static WCO DM Packages (the downloadable spreadsheets, RTF, pdf files), this page is made available to accommodate those usersâ€™ needs where they can download their favourite WCO DM files from the list.

downloadable components will be available for version 3.11.0 and previous versions. Beyond 3.11.0, only those components which do not have an interactive view, such as GOVCBR EDIFACT messages, will be made available as downloadables.
